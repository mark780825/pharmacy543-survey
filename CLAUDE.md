# 藥師決策卡問卷頁面 - 專案設定

## 專案概述
為「社區藥局的五四三 / PHARMACY543」製作的一頁式問卷網頁，仿 Typeform 風格，用於收集藥師對「藥師決策卡」的需求回饋。

## 技術棧
- **前端**: 純 HTML + CSS + Vanilla JavaScript (單一 index.html 檔案)
- **樣式**: 內嵌 CSS，響應式設計，支援手機/平板/桌面
- **表單提交**: Google Apps Script Web App (POST JSON → Google Sheets)
- **部署**: 靜態 HTML，可直接在任何主機託管

## 問卷結構
共 13 題，分 4 大部分：
1. **第一部分：基本背景** (Q1-Q2)
2. **第二部分：第一線的困境** (Q3-Q6)
3. **第三部分：對「藥師決策卡」的期待** (Q7-Q9)
4. **第四部分：了解藥師決策卡方案** (Q10-Q13)

## Google Sheets 欄位對應 (共 15 欄)
| 欄位 | 內容 |
|------|------|
| A | Q1 身分 |
| B | Q2 執業年資 |
| C | Q3 卡住情境 (逗號分隔) |
| D | Q4 處理方式 |
| E | Q5 健康主題 (逗號分隔) |
| F | Q6 查詢方式 |
| G | Q7 決策卡期望 (逗號分隔) |
| H | Q8 最不能接受 |
| I | Q9 自由填寫 |
| J | Q10 方案選擇 |
| K | Q11 Email |
| L | Q12 願意收通知 |
| M | Q13 一句話期望 |
| N | Submitted At |
| O | Token |

## 設計規範
- 品牌色：綠色系 (#2E7D6F 主色)，搭配淺灰背景
- 字體：系統字體優先，確保中文顯示良好
- 動畫：平滑滾動、漸入效果、進度條
- 必填驗證：Q1-Q8, Q10 為必填 (*標記)
- Q3 限選 3 項、Q5 限選 3 項、Q7 限選 2 項

## 禁止修改清單
- 問卷題目文字內容（與原始 Typeform 一致）
- Google Sheets 欄位順序
- 品牌名稱「社區藥局的五四三」
